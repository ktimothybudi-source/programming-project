 BIT Anomalies – 夜市 (Night Market) Assets Placeholder
 ------------------------------------------------------
 
 This folder is intentionally left without real assets so that the project
 compiles and runs out of the box using only procedural / placeholder visuals.
 
 You may add your own assets later, for example:
 
 - audio/night_market_bgm.ogg
 - audio/anomaly_ping.wav
 - textures/...
 
 The code is written so that missing files will NOT crash the game.
